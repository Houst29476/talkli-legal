# talkli-legal

Public-facing legal + support pages for [Talkli](https://talkli.app).

- [Privacy Policy](privacy.md)
- [Support](support.md)

Hosted via GitHub Pages.

## Editing

These pages are written in plain Markdown. To update:

1. Edit the file
2. Commit + push to `main`
3. GitHub Pages rebuilds automatically (~1 min)

For substantive privacy policy changes, update the "Last updated" date at the top of the file and notify users in-app before the change takes effect.
